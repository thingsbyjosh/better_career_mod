# Better Career Mod (BCM)

An open-world career overhaul for [BeamNG.drive](https://www.beamng.com/) — start in an empty garage and progress realistically.

## Features

- **Realistic Economy** — Banking with credit scores, loans, mortgages, and transaction ledgers
- **Vehicle Marketplace** — Curated + procedurally generated listings with negotiation, defects, and scam detection
- **PlanEx Courier System** — Multi-stop delivery dispatch with 20-level driver progression
- **Police & Pursuits** — 3-level escalation system with interceptors, radar, spike strips, and heat tracking
- **Property System** — Buy, upgrade, and manage garages across maps
- **Dynamic Weather** — Seasonal weather with smooth transitions and physics effects
- **Phone & Apps** — Full mobile interface with banking, marketplace, chat, email, and more
- **XP Desktop Browser** — Windows XP-inspired in-game browser with multiple themed websites
- **Time System** — Configurable day/night cycles with sleep mechanics
- **Tutorial System** — 7-step guided introduction for new players
- **Bilingual** — Full English and Spanish support

## Installation

1. Download the latest release (`better_career_mod.zip`)
2. Place the zip file in:
   ```
   %LOCALAPPDATA%\BeamNG.drive\current\mods\
   ```
3. Launch BeamNG.drive and start a new career

## Supported Maps

- West Coast USA
